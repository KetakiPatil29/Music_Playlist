# 🎶 Music Playlist

A web application to explore and enjoy Bollywood music playlists with integrated streaming options like **Spotify** and **YouTube**. Designed with a clean UI and responsive layout for seamless listening.

---

## 🚀 Features
- 🎧 Play embedded Bollywood songs directly from Spotify and YouTube
- 📀 Card‑based layout for easy browsing
- 🌐 Responsive frontend design
- 🔗 Integration with multiple streaming platforms

---

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript, ReactJS
- **Backend:** Java, Spring Boot
- **Database:** MySQL / PostgreSQL
- **Version Control:** Git & GitHub
